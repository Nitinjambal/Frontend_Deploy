# Todo App
Todo App- In this website we can priorities our work just by simply adding our tasks here


# Features
* Navbar
* Login/Logout (with jwt token and bcrypt)
* Register 
* Show Profile
* Read task
* Add task
* Update task
* Delete task



# Tech Stack
frontend:- 
* React
* Router-router-dom
* Chakra-Ui
* NPM packages

Backennd:-
* MongoDB
* Mongoose
* Express
* Nodejs
* Jwt 
* Postman
